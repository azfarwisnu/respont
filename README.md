## Installation
<h1>Front End</h1>

```shell
npm install package.json
npm start
```

<h1>Back End</h1>

```shell
cd backend
npm install package.json
nodemon app.js
```

## Usage
1. Install metamask extension for wallet https://metamask.io/
2. connect wallet for start chat
